# README

In this project I'm creating an HR emplyee system using ruby on rails.

The user interface has been updated with Bootstrap. The video shows all the features of the HR system:

- Add user
- Add an employee
- Add a team
- Add a section
- Add goals

- Edit user
- Edit employee
- Edit the team
- Edit Division
- target adjustment

- Update user
- Update employee
- Update the team
- Update Division
- Update adjustment


- Register a new user
- User logout
- User login

- Change the language from English to Arabic
